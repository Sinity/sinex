## Prelude Imports (USE for common imports)

```rust
// Recommended: flattened imports for most code
use sinex_primitives::prelude::*; // Event, Id, SinexError, validation, domain types
use sinex_db::prelude::*;         // DbPool, DbPoolExt, repositories

// Test utilities (via xtask sandbox feature in tests)
#[cfg(test)]
use xtask::sandbox::prelude::*;   // TestContext, sinex_test, Timeouts, etc.
```

## Event Creation

**Typed Payloads (preferred):** Use fluent methods on `EventPayload` types.

```rust
use sinex_primitives::types::events::EventPayload;
use sinex_primitives::types::events::payloads::FileCreatedPayload;

// Material provenance (ingestors - raw source data)
let event = FileCreatedPayload { path: "/file.txt".into(), size: 1024, ... }
    .from_material(source_material_id)
    .build()?;

// Synthesis provenance (automatons - derived from other events)
let event = AnalyticsSummaryPayload { ... }
    .from_parents(parent_event_ids)?
    .build()?;

// Direct construction with explicit provenance
let event = payload.into_event(provenance);
```

**Dynamic Events (escape hatch):** For JSON payloads without typed structs.

```rust
use sinex_primitives::types::EventBuilder;

// Use dynamic() only for JsonValue - requires explicit source/type
let event = EventBuilder::dynamic("source", "event.type", json!({...}))
    .from_material(material_id, anchor_byte)
    .build()?;
```

## Error Handling | ALWAYS USE SinexError

```rust
use sinex_primitives::types::error::{SinexError, Result};

// Create errors with context
SinexError::validation("Invalid input")
    .with_context("field", "username")
    .with_context("reason", "too short")

// Use #[with_context] macro for automatic enrichment
#[with_context]
fn process_data(input: &str) -> Result<Output> {
    // Errors automatically include function name, module
}
```

## Database Access (USE Repository Pattern)

```rust
use sinex_db::DbPoolExt;

// Correct: Use repository methods
let events = pool.events().get_by_source(&source, pagination).await?;
let blob = pool.blobs().get_by_id(id).await?;

// Available repositories:
pool.events()           // EventRepository
pool.blobs()            // BlobRepository
pool.source_materials() // SourceMaterialRepository
pool.knowledge_graph()  // KnowledgeGraphRepository
pool.state()            // StateRepository
pool.schemas()          // SchemaManagementRepository
pool.schema_cache()     // SchemaCacheRepository - Schema lookup & caching
```

## Domain Types (USE instead of raw strings)

```rust
use sinex_primitives::types::domain::*;

let source = EventSource::new("fs-watcher");      // Not String (usually use autogenerated constants)
let event_type = EventType::new("file.created");  // Not String (usually use autogenerated constants)
let hostname = HostName::new("myhost");           // Not String

// Compile-time type safety: can't mix EventSource with EventType
```

## Typed IDs (USE Id<T>)

```rust
use sinex_primitives::types::Id;
use sinex_primitives::types::models::Event;

let event_id: Id<Event<JsonValue>> = Id::new();  // ULID-based, type-safe
let uuid = event_id.to_uuid();                    // For database
let ulid = event_id.as_ulid();                    // For display
```
