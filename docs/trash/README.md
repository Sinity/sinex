# docs/trash

Purpose
- Temporary quarantine for redundant, superseded, or illegitimate docs during cleanup.
- Allows safe review before permanent removal.

Policy
- Keep authoritative specs in `docs/architecture/` and top‑level `docs/` only.
- Move snapshots/audits/long‑form narratives here when they no longer reflect reality.
- Salvage any reusable guidance into canonical docs; then delete the originals.

Retention
- Short‑lived: files should be removed after consolidation (typically same PR).
- In `main`, this directory should remain empty (aside from this README).
- Retrieve prior content via Git history if needed.

Status
- As of this cleanup, prior contents have been fully retired. See Git history for details.
