# Archive

This folder holds legacy documentation preserved for historical reference. Documents here are frozen: they are not maintained, and may be inaccurate.

Policy
- Keep the main docs clean; move old analyses/snapshots here when their useful content has been distilled into canonical docs.
- Do not link new materials to files in `archive/` — prefer to cite the updated sources.
- Retrieve older content via Git history where possible.

Structure
- `analysis-old/` — older analysis reports previously under `docs/analysis/old/`.
- `tmp_store/` — staging artifacts from earlier migrations.
