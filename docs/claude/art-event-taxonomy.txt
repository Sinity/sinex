                          THE PROVENANCE PRINCIPLE

            Every event carries its lineage:

            ┌─────────────────────────────────────────┐
            │  ULID: 01HQXYZ...                       │  Primary event
            │  Type: file.created                     │  (Raw observation)
            │  Payload: {path: "/docs/thesis.md"}     │
            │  Source: ingestor.filesystem            │
            └──────────────────┬──────────────────────┘
                               │
                               ▼
            ┌─────────────────────────────────────────┐
            │  ULID: 01HQXZA...                       │  Derived event
            │  Type: document.analyzed                │  (Synthesis)
            │  Payload: {word_count: 5432}            │
            │  Source: automaton.content              │
            │  source_event_ids: [01HQXYZ...]         │  ← Provenance!
            └──────────────────┬──────────────────────┘
                               │
                               ▼
            ┌─────────────────────────────────────────┐
            │  ULID: 01HQXZB...                       │  Higher-order
            │  Type: knowledge.entity_extracted       │  (Knowledge)
            │  Payload: {entity: "Thesis Topic"}      │
            │  Source: automaton.pkm                  │
            │  source_event_ids: [01HQXZA...]         │  ← Chain intact
            └─────────────────────────────────────────┘
